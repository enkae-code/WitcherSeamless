# WitcherSeamless Tools

This directory contains production deployment tools for the WitcherSeamless multiplayer system.

## Tools

- **w3m.exe** - Automated multiplayer client launcher
  - Handles game startup and multiplayer injection
  - Provides seamless integration with Witcher 3
  - Replaces manual game launching

## Usage

The `Launch_W3MP.bat` script in the project root automatically uses tools from this directory.
